# Django Basic


## Setup environment 🏗
```cmd
>> pip install django
>> django-admin startproject <project_name>
>> django-admin startapp <app_name>
```

## Running server 🏎
```cmd
>> py manage.py runserver
```


## Handling migrations 🧩
```cmd
>> py manage.py makemigrations
>> py manage.py migrate
```

## Create adm access 👑
```cmd
>> py manage.py createsuperuser
```
